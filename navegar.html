Cómo usar la web

Este sitio ha sido diseñado para incorporar conceptos de programación estructurada a partir de la implementación de ejemplos y su análisis, para luego introducir los conceptos, estructuras e instrucciones requeridas.
Para ello, en las distintas secciones hay código que puedes copiar, para luego pegar y ejecutar en PSeInt.
También hay secciones llamadas "jugando con PSeInt" donde se plantean cuestionamientos acerca de cómo se comporta el código desarrollado.


Para poder implementar el código del sitio, debemos tener instalado PSeInt.

Las distintas secciones están separadas por pestañas en cinta superior (PC) o botón menú (celular). Contenido de las pestañas:


Saludos: cuestiones introductorias, estructura secuencial.
Esto o aquello: sobre estructuras condicionales.
Repetir ayuda: sobre estructuras repetitivas, particularmente repetitiva Para.
Repetir hasta que: Sobre repetitiva Repetir-hasta.
